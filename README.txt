Voici le projet du Mini RPG. Pour le tester, il faut ouvrir le fichier config.html. Entrez un pseudo et validez-le,
choisissez ensuite votre personnage et votre arène puis cliquez sur lancer le jeu. La page du jeu s'ouvrira dans un
nouvel onglet.
